# MH 5

Monochrome variant of the MH theme. Features grayscale accents with the blood red chat bubble as the only color element. Minimalist aesthetic for distraction-free coding.

**Accent Color**: Grayscale (#a0a0a0, #c0c0c0, #d0d0d0)

Customized elements:

- Cursor, tab borders, and panel borders in neutral gray
- List selections and highlights in grayscale
- Bracket pair colorization in gray tones
- Blood red chat sent bubble (#8b0000) remains as sole color accent
