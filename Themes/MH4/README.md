# MH 4

Blue/Cyan accent variant of the MH theme. Features bright blue highlights while maintaining the blood red chat bubble and dark UI foundation.

**Accent Color**: Blue/Cyan (#3b82f6, #60a5fa)

Customized elements:

- Cursor, tab borders, and panel borders in bright blue
- List selections and highlights in blue shades
- Bracket pair colorization includes cyan tones
- Maintains blood red chat sent bubble (#8b0000)
