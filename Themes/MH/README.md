# MH

Primary dark theme (formerly MattioliHoffman) with blood-red chat request bubble and dark teal input.

## Key Characteristics

- Neutral dark base (#1a1a1a) with near-black chrome accents.
- Blood red chat request bubble (#8b0000 / hover #c41e1e).
- Dark teal input & quick input (#08201d) for focus without glare.
- Teal/seafoam accent (#23bfa5) for cursor, borders, highlights.
