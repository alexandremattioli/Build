# MH (Teal)

Teal variant (formerly MattioliHoffman (Teal)) where the chat request bubble is a deep teal instead of red.

## Key Characteristics

- Deep teal chat request bubble (#0f3d34 / hover #144a3f).
- Dark teal input & quick input (#08201d); bright neutral text.
- Teal/seafoam accent (#23bfa5) for focus elements.
